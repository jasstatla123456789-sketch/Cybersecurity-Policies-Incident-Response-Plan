# Purpose

The purpose of this Acceptable Use Policy (AUP) is to establish the rules and responsibilities for the proper use of the organization's information technology resources. These resources include computers, laptops, mobile devices, networks, internet access, cloud services, email systems, software, and company-owned data.

Following this policy helps protect business information, reduce cybersecurity risks, and ensure that technology resources are used responsibly and ethically.

---

# Scope

This policy applies to:

- Full-time employees
- Part-time employees
- Interns
- Contractors
- Consultants
- Third-party vendors
- Anyone using company IT resources

---

# Acceptable Use

Employees are expected to use company technology resources only for authorized business purposes.

Users may:

- Access business applications required for their job.
- Use company email for official communication.
- Access approved cloud services.
- Participate in online meetings.
- Store business documents in approved locations.
- Access the internet for work-related research.
- Use company devices in accordance with organizational policies.

---

# Prohibited Activities

Users must not:

- Install unauthorized software.
- Download pirated or illegal content.
- Disable antivirus or firewall protection.
- Share usernames or passwords.
- Attempt to bypass security controls.
- Connect unauthorized devices to the company network.
- Access inappropriate or offensive websites.
- Use company resources for illegal activities.
- Perform unauthorized hacking or penetration testing.
- Modify system configurations without approval.

---

# Internet Usage

Employees should use the internet responsibly.

Allowed activities include:

- Business research
- Vendor communication
- Cloud-based collaboration
- Professional development

The following activities are prohibited:

- Online gambling
- Streaming copyrighted media without authorization
- Visiting malicious or unsafe websites
- Downloading unknown files
- Accessing illegal content

---

# Email Usage

Employees must:

- Use company email only for business purposes.
- Verify unknown senders before responding.
- Report phishing emails immediately.
- Avoid opening suspicious attachments.
- Never share confidential information without authorization.

---

# Password Requirements

Users must:

- Create strong passwords.
- Keep passwords confidential.
- Enable Multi-Factor Authentication (MFA) where available.
- Report suspected credential compromise immediately.
- Never write passwords where unauthorized individuals can access them.

---

# Software Installation

Only the IT Administrator may:

- Install software.
- Remove software.
- Update operating systems.
- Configure security software.

Employees must obtain approval before requesting additional applications.

---

# Data Protection

Employees must:

- Protect confidential business information.
- Store sensitive data only in approved locations.
- Encrypt confidential files when required.
- Lock computers when unattended.
- Properly dispose of confidential documents.

---

# Mobile Device Usage

Company mobile devices must:

- Use screen locks.
- Enable device encryption.
- Receive regular software updates.
- Install approved security software.
- Be reported immediately if lost or stolen.

---

# Remote Access

Employees accessing company resources remotely must:

- Use the company VPN.
- Enable MFA.
- Connect through secure Wi-Fi networks.
- Avoid using public computers.
- Keep operating systems updated.

---

# Monitoring

The organization reserves the right to monitor:

- Internet activity
- Email usage
- Network traffic
- File access
- System logs
- Security alerts

Monitoring is performed solely for security, compliance, and operational purposes.

---

# Reporting Security Incidents

Employees must immediately report:

- Suspicious emails
- Malware infections
- Lost devices
- Unauthorized access
- Data leaks
- Any suspected cybersecurity incident

Reports should be submitted to the IT Administrator without delay.

---

# Employee Responsibilities

Employees are responsible for:

- Following this policy.
- Protecting company assets.
- Reporting security concerns.
- Completing cybersecurity awareness training.
- Using company resources responsibly.

---

# Policy Violations

Failure to comply with this policy may result in:

- Verbal warning
- Written warning
- Suspension of system access
- Disciplinary action
- Termination of employment
- Legal action where applicable

---

# Policy Review

This policy shall be reviewed:

- Annually
- Following significant cybersecurity incidents
- After major technology changes
- When regulatory requirements change
